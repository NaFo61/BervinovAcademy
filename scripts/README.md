# Скрипты проекта Bervinov Academy

Эта папка содержит вспомогательные скрипты для разработки и администрирования проекта.

## 📋 Доступные скрипты

### 1. Генерация промта проекта

**Файлы:**
- `generate-prompt.py` - основной Python скрипт
- `generate-prompt.bat` - обертка для Windows
- `generate-prompt.sh` - обертка для Linux/Mac
- `project_prompt.txt` - результат работы скрипта (генерируется автоматически)

**Назначение:**
Скрипт собирает информацию о проекте и генерирует подробное описание (промт), которое можно использовать для AI-ассистентов (например, Cline, Cursor, GitHub Copilot).

**Что включает промт:**
- Общую информацию о проекте (название, описание, тип)
- Основные возможности платформы
- Технологический стек (backend, frontend, infrastructure)
- Список Django приложений
- Структуру проекта с иконками типов файлов
- Содержимое ключевых файлов (docker-compose.yml, README.md, pyproject.toml и др.)
- Конфигурацию Docker Compose сервисов

**Использование:**

```bash
# Windows (через .bat файл)
scripts\generate-prompt.bat

# Linux/Mac (через .sh файл)
./scripts/generate-prompt.sh

# Напрямую через Python
python scripts/generate-prompt.py
```

**Результат:**
Скрипт автоматически сохраняет промт в файл `scripts/project_prompt.txt` и выводит его на экран.

**Пример вывода:**

```
================================================================================
ПРОМТ ПРОЕКТА: Bervinov Academy
================================================================================

📋 ОБЩАЯ ИНФОРМАЦИЯ
----------------------------------------
Название: Bervinov Academy
Версия: 0.1.0
Описание: Современная образовательная платформа с менторской поддержкой...
Тип: Fullstack веб-приложение (Django + React)
Сгенерировано: 2026-04-30T11:42:09.789867

⭐ ОСНОВНЫЕ ВОЗМОЖНОСТИ
----------------------------------------
1. Структурированные курсы с модулями и уроками
2. Менторская поддержка для каждого студента
...
```

**Зачем это нужно:**
- Быстро предоставить AI-ассистенту полный контекст проекта
- Автоматически обновлять описание проекта при изменениях
- Использовать как основу для генерации документации
- Помочь новым разработчикам быстро понять структуру проекта

## 📝 Заметки

1. Скрипты `.bat` и `.sh` являются обертками вокруг Python скрипта `generate-prompt.py` и обеспечивают удобный запуск.
2. При запуске скрипты автоматически определяют корневую директорию проекта.
3. Сгенерированные промты можно использовать в AI-ассистентах для получения более точных ответов.
4. Для Linux/Mac не забудьте сделать скрипты исполняемыми: `chmod +x scripts/generate-prompt.sh`
